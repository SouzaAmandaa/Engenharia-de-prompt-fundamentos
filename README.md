# Engenharia-de-Prompt-e-Aplicacao-AI

## Comparação pratica de prompts 

### Prompt Genérico

* Pergunta: * O que é Oracle Apex?
  
* Resposta: * Oracle APEX (Oracle Application Express) é uma plataforma de desenvolvimento low‑code da Oracle usada para criar aplicações web de forma rápida, segura e com pouco ou nenhum código.

### Prompt Estruturado 

* Prompt utilizado: * Atue como professor e especialista em sistemas low code. Explique Oracle Apex para um iniciante, usando uma analogia com esportes, em 2 páragrafos.
  
* Resposta: * O Oracle APEX é como uma escola de esportes para iniciantes, onde você não precisa ser um atleta profissional para começar a jogar. Imagine que você vai aprender basquete: em vez de te jogarem direto num campeonato difícil, a escola te oferece bolas leves, uma quadra simples e treinos guiados. O APEX faz a mesma coisa no mundo dos sistemas: ele te entrega ferramentas prontas, modelos e recursos fáceis de usar para que você desenvolva aplicações web mesmo sem ser um programador experiente. Você só precisa de um navegador, assim como só precisa de tênis e uma quadra para começar a treinar.
Agora pense no time: em alguns esportes, como futebol, você passa a bola, usa estratégias simples e o jogo flui naturalmente. No APEX, o “time” são o banco de dados e os componentes prontos — formulários, gráficos, relatórios — que já sabem trabalhar juntos. Você apenas define as regras do jogo e o APEX executa. Assim como um técnico cria jogadas usando habilidades básicas dos jogadores, você cria sistemas combinando os elementos visuais com o poder do banco de dados Oracle, sem ter que programar tudo do zero.

## Análise Crítica 

A resposta mais útil foi a segunda, de um prompt estruturado. A estrutura do prompt fez com que a resposta fosse menos génerica e ambígua, podendo dessa forma ajudar quem ainda esta iniciando no assunto para que o entendimento seja simples e claro.
